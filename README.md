# JPHACKS2022_b2212
JPHACKSに向けて開発中の初心者向け機械学習コード生成ツール

## development
1. git clone git@github.com:ReoYabiku/JPHACKS2022_b2022.git
1. npm install
1. docker compose build --no-cache
1. docker compose up

- バックエンドは5000ポート、フロントエンドは3000ポートからアクセスできる。
